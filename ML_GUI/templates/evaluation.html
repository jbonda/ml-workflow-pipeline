Model Evaluation

Model evaluation is the process of using different evaluation metrics to understand a machine learning model's performance, as well as its strengths and weaknesses. Here we are using the following metrics:

  • Root Mean Squared Error (RMSE) â†’ Root Mean Squared error is a popular metric used in machine learning to measure the accuracy of a predictive model.It quantifies the difference between predicted values and actual values, squaring the errors, take the mean, and then finding the square root.
    \[ RMSE = \sqrt{\frac{1}{N}\sum_{i=1}^{N} (y_i - \hat{y}_i)^2} \]
    ... where N is the number of data points, \(y_i\) is the i-th measurement, and \(Å·_i\) is its corresponding prediction.

  • Accuracy Score â†’ Accuracy Score is one metric for evaluating classification models. Accuracy has the following definition:
    \[ \text{Accuracy} = \frac{\text{Total Number of Predictions}}{\text{Number of Correct Predictions}} \]
    For binary classification, accuracy can also be calculated in terms of positives and negatives as follows:
    \[ \text{Accuracy} = \frac{\text{TP + TN}}{\text{TP + TN + FP + FN}} \]
    ... Where TP = True Positives, TN = True Negatives, FP = False Positives, and FN = False Negatives.

  • Confusion Matrix â†’ A Confusion matrix is an N x N matrix used for evaluating the performance of a classification model, where N is the total number of target classes. The matrix compares the actual target values with those predicted by the machine learning model.
