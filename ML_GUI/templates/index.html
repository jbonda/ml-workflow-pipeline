Machine Learning-GUI Web Application

Welcome to the Machine learning web app!

This website is designed to train a data set by Linear Regression, Polynomial Regression, and Logistic Regression.

The regression can be defined as:

"Using the relationship between variables to find the best-fit line or the regression equation that can be used to make predictions." -- Towards Data Science

Data Upload

Upload a data file. Make sure your file extension is .CSV (comma-separated values).

Split Data

Splitting the data file into a train data set and a test data set by choosing the test data value. Test data should less than train data for an optimum model.
